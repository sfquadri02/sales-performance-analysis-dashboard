# sales-performance-analysis-dashboard
Power BI dashboard analyzing sales performance and trends

## Project Overview
This project analyzes sales performance data using Power BI Desktop to track
revenue trends, category performance, and overall business metrics.

## Business Questions Addressed
- How is sales performance trending over time?
- Which product categories contribute the most to revenue?
- What patterns can be identified in sales data?

## Tools and Technologies
- Power BI Desktop
- Microsoft Excel
- DAX
- Data Modeling
- Exploratory Data Analysis (EDA)

## Key Insights
- Identified overall sales performance trends
- Analyzed category-level contribution to total revenue
- Highlighted key performance patterns to support decision-making
